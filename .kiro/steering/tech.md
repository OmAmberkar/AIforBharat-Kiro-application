# Technical Stack & Guidelines

## Stack
* **Frontend:** Pure HTML5, CSS3, Vanilla JavaScript (ES6+).
* **Hosting:** GitHub Pages.
* **Styling:** Custom CSS with CSS Variables for theme consistency. No external frameworks (Bootstrap/Tailwind) to keep it lightweight.

## Constraints
* No build steps (npm/yarn) required.
* Must be responsive for mobile usage.
* Clean, semantic HTML structure.
